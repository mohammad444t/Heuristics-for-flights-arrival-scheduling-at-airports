To install CPLEX (on windows):
1. Download CPLEX From https://cdna.p30download.ir/p30dl-software/IBM.ILOG.CPLEX.Optimization.Studio.v22.1.0.x64_p30download.com.rar
2. Install CPLEX on windows
3. Open terminal in your venv
4. Type python "C:\Program Files\IBM\ILOG\CPLEX_Studio221\python\setup.py" install and press enter
5. pip install cplex
6. pip install docplex
You are done!
